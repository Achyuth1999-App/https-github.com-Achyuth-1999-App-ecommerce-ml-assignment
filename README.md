
  




 E-Commerce Marketing and Product Performance Analysis

 Project Overview
This project applies supervised and unsupervised machine learning techniques to analyze a global e-commerce dataset. The goal is to predict revenue, classify customers, and understand key factors influencing sales performance such as pricing, discounts, product categories, and regions.

---

 Dataset Information
- Dataset Name: Global E-Commerce Dataset (1M Records)
- Source: Kaggle
- Link: https://www.kaggle.com/datasets/akrambelha/global-e-commerce-dataset-1m-records
- Size: ~1,000,000 rows

---

 Objectives
- Predict revenue using regression models  
- Identify customer segments using clustering  
- Classify high-value customers  
- Analyze impact of pricing, discounts, category, and region  

---

 Research Questions
1. Can we predict revenue using product and pricing features?  
2. What customer segments exist?  
3. Which product categories perform best?  
4. How does region affect sales?  
5. Can we classify high-value customers?  
6. How does price impact revenue?  
7. How do discounts affect sales?  

---
 Methodology
1. Data Cleaning  
2. Feature Engineering  
3. EDA  
4. Model Building  
5. Evaluation  
6. Visualization  

---

 Models
- Linear Regression  
- Random Forest  
- Logistic Regression  
- K-Means  

---

Metrics
- Regression: MAE, RMSE  
- Classification: Accuracy, Precision, Recall  
- Clustering: Silhouette Score
